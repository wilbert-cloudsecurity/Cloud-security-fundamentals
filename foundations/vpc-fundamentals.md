# AWS VPC Fundamentals

## What VPC Means
Amazon Virtual Private Cloud (VPC) allows you to create an isolated virtual network inside AWS where you can launch and manage resources securely.

## Why VPC Matters
VPC is the foundation of network architecture in AWS. It controls how resources communicate internally and with the internet.

---

# Core Components

## CIDR Block
- Defines the IP address range of the VPC
- Example: 10.0.0.0/16

## Subnets
- Logical subdivisions of a VPC
- Can be public or private
- Distributed across Availability Zones

## Route Tables
- Control traffic routing within the VPC
- Determine internet or internal access

## Internet Gateway
- Enables communication between VPC resources and the internet
- Attached to the VPC
