# Cloud Security Fundamentals

This repository documents my learning journey toward a career in **Cloud and Cloud Security**.
It focuses on building strong fundamentals, hands-on practice, and consistent documentation
of progress over time.

The goal of this repository is to reflect real learning and practical understanding,
not just completed tasks.

---

## 📂 Repository Structure

- `foundations/`  
  Core technical foundations such as Linux, Git, and cloud fundamentals.

- `monthly-progress/`  
  Monthly summaries describing what was learned, practiced, and reinforced.

- `shared-responsibility.md`  
  Notes and explanations of the Cloud Shared Responsibility Model,
  with emphasis on security boundaries and responsibilities.

- `README.md`  
  Repository overview and context.

- `mini-projects`
  collection of exercises and projects to apply theory into practice.
